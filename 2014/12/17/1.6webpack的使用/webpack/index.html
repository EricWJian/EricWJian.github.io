1.使用 webpack-dev-server 这个工具，来实现自动打包编译的功能
1.npm install webpack-dev-server -D
依赖于webpack 在本地安装webpack
错误提示 缺少webpack-cli，安装上即可
2.安装 html-webpack-plugin
作用：1. 自动在内存中根据指定页面生成一个内存的页面

          2. 自动，把打包好的 bundle.js 追加到页面中去
3.处理样式
    安装 style-loader 和 css-loader
4.url-loader（依赖于file-loader） 处理url图片
