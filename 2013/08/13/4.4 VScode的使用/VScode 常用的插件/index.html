VScode 常用插件

发表于2013-08-13|更新于2022-03-19
|阅读量:

配置类插件

1.code settings sync

2.Debugger for Chrome

3.beautify

4.Atuo Rename Tag

修改 html 标签，自动帮你完成头部和尾部闭合标签的同步修改

5.Chinese (Simplified) Language Pack for Visual Studio Code

6.guides

显示代码对齐辅助线，很好用

7.Rainbow Brackets

为圆括号，方括号和大括号提供彩虹色。这对于Lisp或Clojure程序员，当然还有JavaScript和其他程序员特别有用。

8.Bracket Pair Colorizer

用于着色匹配括号

9.Indent-Rainbow

用四种不同颜色交替着色文本前面的缩进
