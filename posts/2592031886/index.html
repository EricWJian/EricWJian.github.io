6、安装插件

先去手动安装一个插件：skip-certificate-check，用于ssl无法识别问题以及插件安装超时问题。
插件可以去Jenkins官网的插件中心直接下载，但是可能会打不开网站或者打开非常慢，
也可以到国内清华大学开源软件镜像站的插件中心下载（建议），插件是一个hpi后缀名的文件，下载后直接放在你的Jenkins的plugins目录下即可。

http://mirror.esuni.jp/jenkins/updates/update-center.json

1、Git Parameter

这是一个参数构建扩展，可以在构建的时候选择git的某一个分支来构建服务。

2、Publish Over SSH

通过SSH拷贝文件到目标机器，同时可以在目标机器上执行脚本。

3、GitLab

5、war 包方式安装

  1. 下载 jenkins.war 包
  2. nohup java -jar jenkins.war >> jenkins.log 2>&1 &
  3. 访问地址：http://192.168.58.201:8080/jenkins/
  4. rm -rf /root/.jenkins 后，Jenkins会重新初始化
