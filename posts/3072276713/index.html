一、 简介

Spring Boot Admin是一个开源社区项目，用于管理和监控SpringBoot应用程序。

Spring Boot Admin Server 作为监控服务端， 要监控所有的客户端的运行状态， 实际是通Actuator的/health监控节点来获取服务状态， 客户端和服务端都设置一个任务监听器，定时监听对方的心跳，并在服务器及时更新客户端状态。

Prometheus + Grafana：也是一套可视化的监控平台。
