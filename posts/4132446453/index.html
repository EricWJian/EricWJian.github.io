自动拉取构建

在 GitLab 上创建一个项目，并将代码上传到仓库进行管理。

1、全局工具配置

Dashboard>Manage Jenkins>Global Tool Configuration

1、配置 jdk

项目打包，需要jdk。

当输入框失去焦点时，如果不报错，证明配置正确。

2、配置 Git

Git用于拉取远端代码。

当输入框失去焦点时，如果不报错，证明配置正确。

3、配置maven

maven 用于打包。
