Node 使用

发表于2018-03-13|更新于2023-09-21|前端
|字数总计:916|阅读时长:3分钟|阅读量:

nodejs通俗的理解: 一种javascript的运行环境，能够使得javascript脱离浏览器运行。

一、Node 的安装

1、普通安装

不推荐:

普通安装方式[官方网站] （无法进行node版本升级）

2、多版本安装

多版本安装方式，nvm 全称 Node Version Manager是 Nodejs 版本管理器，它让我们能方便的对 Nodejs 的版本进行切换。 nvm 的官方版本只支持 Linux 和 Mac。 Windows 用户，可以用 nvm-windows。

安装方式

  1. 卸载已有的Node.js

  2. 下载[nvm] 推荐安装 nvm-setup.zip

  3. 在 C 盘创建目录dev

  4. 在 dev 目中中创建两个子目录 nvm 和 nodejs

  5. 并且把 nvm 包解压进去nvm目录中

  6. 在 install.cmd 文件上面右键选择【以管理员身份运行】

  7. 打开的 cmd 窗口直接回车会生成一个 settings.txt 文件，修改文件中配置信息

  8. 配置nvm和Node.js环境变量

  9. 把配置好的两个环境变量加到 Path 中 %NVM_HOME%;%NVM_SYMLINK%

  10. 测试 nvm

  11. 配置镜像加速,settings.txt 文件中添加

二、nvm常用的命令

  1. node -v 查看node的版本

  2. nvm list 查看当前安装的Node.js所有版本

  3. nvm install 版本号 安装指定版本的Node.js（latest 最新）

  4. nvm uninstall 版本号 卸载指定版本的Node.js

  5. nvm use 版本号 选择指定版本的Node.js

三、NPM 和 NRM

1、npm

Nodejs 下的包管理器(Node Package Manager ,随同 NodeJS 一起安装的包管理工具，新版的 nodejs 已经集成了npm，所以 npm 也一并安装好了。）

  • npm -v 查看是否安装成功

2、nrm

nrm工具(npm registry manager , nrm是一个 npm 源管理器，允许你快速的在 npm源之间切换，但是我们每次装包的时候，使用的工具都是npm）

  1. 全局安装nrm包

  2. 查看当前所有可用的景象地址以及当前所使用的镜像地址

  3. 测试镜像速度

  4. 切换不同的镜像源地址

3、cnpm

cnpm 装包工具(npm 插件是从国外服务器下载的，受网络影响较大，可能出现异常，cnpm 拥有完整的 npm 镜像，10 分钟同步有一次)，使用方式 和 npm 一样；但是不推荐使用 cnpm 。

  • npm i cnpm -g
